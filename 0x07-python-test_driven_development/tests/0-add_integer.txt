The ``0-add_integer`` module
============================

Using ``add_integer``
---------------------

Import example first then use it

    >>> add_integer = __import__('0-add_integer').add_integer

Usage:

    >>> add_integer(10, 20)
    30
    >>> add_integer(2, -3)
    -1

The function also works with floats:
    
    >>> add_integer(1.0, 2.0)
    3

Floats are casted to integers before operation is done

    >>> add_integer(2.9, 0.2)
    2

    >>> add_integer(-2.9, -0.2)
    -2
